## 2. データモデル

### アイデア (Idea)

```
{
  id:     number,          // 連番
  x:      number,          // ボード座標 (px)
  y:      number,
  text:   string,          // 本文（マークダウン風、1行目=タイトル）
  color:  ColorName,       // 15色のいずれか（デフォルト: 'purple'）
  layer:  number,          // 所属レイヤーID（1〜3）
  shape:  'rect'|'pill',   // 形状（省略時 'rect'）。'pill'=横長の丸（スタジアム形）
}
```

### ゾーン (Zone)

```
{
  id:     string,          // 'z' + 連番
  x, y:  number,
  w, h:  number,           // 幅・高さ (px)
  label:  string,          // 上部ラベル
  color:  ColorName?,
  layer:  number,
}
```

### コネクター (Connector)

```
{
  id:   string,
  from: { kind: 'idea'|'zone', id },
  to:   { kind: 'idea'|'zone', id },
  type: 'line' | 'arrow' | 'curve' | 'curve-arrow',
}
```

### レイヤー (Layer)

```
{
  id:      number,                          // 1〜3
  name:    string,                          // "Layer 1" 〜 "Layer 3"
  opacity: 'opaque' | 'semi' | 'transparent',
}
```

### 全体状態 (State `S`)

| フィールド | 型 | 説明 |
|---|---|---|
| `ideas` | `{ [id]: Idea }` | 全アイデア |
| `zones` | `Zone[]` | 全ゾーン |
| `connectors` | `Connector[]` | 全コネクター |
| `layers` | `Layer[]` | 3レイヤー固定 |
| `activeLayerId` | `number` | 現在編集中のレイヤー |
| `nextId` | `number` | 次に使うID |
| `pan` | `{x, y}` | ビューポートオフセット |
| `zoom` | `number` | 拡大率（0.25〜2.5） |
| `theme` | `'dark'|'light'` | テーマ |
| `focused` | `{kind, id}?` | 単一フォーカス |
| `moveSelection` | `{kind, id}[]` | 複数選択リスト |

## 4. 機能仕様

### 4.1 アイデア (Idea)

#### 作成

- `i` キー: ビューポート中央に新規作成、即座に編集モードへ
- フォーカス中のアイデアがある場合: その直下（y + height + 20px）にカスケード生成
- 新規アイデアは **アクティブレイヤー** に所属

#### 編集 (`edit` モード)

- `Enter` / ダブルクリック / `Tab` で編集モード開始
- テキストエリア内での操作:
  - `Enter` (単押し): 編集確定 → 直下に次のアイデアを連続生成
  - `Shift+Enter`: 改行（通常のテキスト入力）
  - `Esc` / `Tab`: 編集確定・選択モードへ戻る
- 1行目: タイトルとして大きく表示（Markdown `#` 記号は除去）
- 2行目以降: スニペットプレビューとして小さく表示（最大5行）

#### 表示

- カード幅: `min 192px`, `max 288px`
- 左端アクセントバー: カラーを反映
- ホバー / フォーカス: アクセント色のボーダーハイライト

#### 移動

- ドラッグ: 自由移動（グリッドスナップあり）
- スナップ間隔: 24px（ドットグリッドと同期）
- `Alt` キー押しながらドロップ: スナップなし
- `Shift+Arrow`: 10px 単位ナッジ（フォーカス中）
- ドラッグ中: アライメントガイド（オレンジ点線）で他カードと整列表示

#### 形状変形 (Shape)

- 形状は2種類: `rect`（四角・デフォルト）/ `pill`（横長の丸 = 長方形の左右に半円を付けたスタジアム形）
- `o` キー: フォーカス中アイデア（または複数選択中の全アイデア）の形状を `rect ↔ pill` でトグル
- 複数選択中は上部のセレクションツールバーから「四角」「丸」を明示的に指定可能
- `pill` 形状はボーダー全体にカラーを反映（左端アクセントバーは非表示）

#### 複製

- `Ctrl+D`: フォーカス中アイデアを (+24, +24) オフセットで複製

#### 削除

- `x` / `Delete`: フォーカス中のオブジェクトを削除（接続コネクターも同時削除）

---

### 4.2 ゾーン (Zone)

#### 作成

- `z` キー: ビューポート中央に 280×190px で生成
- 新規ゾーンは **アクティブレイヤー** に所属

#### ラベル編集

- `Enter` / ダブルクリック: ラベルをインライン編集
- 最大30文字

#### リサイズ

- 右下のハンドルをドラッグ
- `r` キー → リサイズモード → `Arrow` キーで 20px 単位変更 (`Shift+Arrow` で 50px)
- `Enter` でリサイズモード終了

#### スタイル

- カラー: 15色（ボーダー・ラベルに適用）
- 角丸: 6px
- 半透明塗りつぶし

---

### 4.3 コネクター (Connector)

#### 接続手順

1. 接続元を選択（フォーカス）
2. `c` キー → コネクトモード開始（接続元がパルスアニメーション）
3. `Arrow` キーまたは `f` キー（Vimium 風ヒント選択）で接続先を選択
4. `Enter`: 直線（ライン）で接続
5. `Shift+Enter`: 矢印付き直線で接続
6. `b` / `Shift+B`: 曲線 / 矢印付き曲線で接続
7. `Esc`: キャンセル

#### 種別 (Type)

| 種別 | 説明 | 描画 |
|---|---|---|
| `line` | 線 | 直線（SVG `L`） |
| `arrow` | 矢印 | 直線 + 矢印マーカー |
| `curve` | 曲線 | 二次ベジェ（SVG `Q`） |
| `curve-arrow` | 曲線の矢印 | 曲線 + 矢印マーカー |

#### 単一コネクタの直接選択（クリック）

- コネクタの線を**クリックすると選択**（フォーカス）できる。細い線でも掴めるよう、見た目の線とは別に幅 14px の透明な当たり判定パスを重ねている
- ホバーで線が太く強調、選択中は太線＋アクセント色のドロップシャドウ
- 選択中は上部ツールバー（コネクタ群＋🗑削除）が表示され、種別変更・`x`/`Delete` 削除が可能
- 右クリックでコンテキストメニュー（種別変更・削除）

#### 種別変更（マウス操作・複数）

1. 対象アイデアを `Shift+クリック` などで複数選択（接続の片側だけでも可）
2. 上部に出るセレクションツールバーで「線 / 矢印 / 曲線 / 曲線矢印」をクリック
3. **選択中アイデアにつながっているコネクタ**（少なくとも一方の端点が選択範囲内）すべての種別が一括変更される

> セレクションツールバーは現在の状態を反映する: 選択中アイデアが共通の形状なら該当ボタンを、対象コネクタが単一種別ならその種別ボタンをハイライト。対象が無いボタンは自動で無効化される。

#### 特性

- プレビュー線は直線（破線・白）
- 同一ペアを再接続: **トグル削除**
- コネクター色: 接続元のカラーを継承
- 矢印マーカー: カラー別に SVG `<marker>` を定義（曲線にも `orient="auto"` で追従）

#### 架け橋 (Bridge)

- `g` キー → Vimium 風で接続先を選択 → 2点間に中継アイデアを自動生成

### 4.4 レイヤーシステム (Layer)

#### UI

ヘッダー右側に常時表示:

```
‹  Layer 1  ›  ●
```

| 要素 | 動作 |
|---|---|
| `‹` ボタン | 前のレイヤーへ切り替え（3→2→1→3 と循環） |
| `›` ボタン | 次のレイヤーへ切り替え（1→2→3→1 と循環） |
| `●` ボタン | アクティブレイヤーの透過設定をサイクル変更 |

#### 透過設定

| 設定 | アイコン | 描画 | インタラクション |
|---|---|---|---|
| `opaque`（不透過） | `●` | 100% | 可 |
| `semi`（半透過） | `◐` | 35% | 可（ただし薄く表示） |
| `transparent`（透過） | `○` | 0%（非表示） | 不可 |

#### 描画ルール

- **アクティブレイヤー**: 設定に関わらず常に 100% 表示・編集可能
- **非アクティブレイヤー**: 各レイヤーの `opacity` 設定に従って描画
- 透過レイヤーの要素: `pointer-events: none`（クリック・ドラッグ不可）
- レイヤー切り替え時: フォーカスをクリア、全要素のopacityを即時更新

#### データ

- レイヤー数: 固定 3（Layer 1 / Layer 2 / Layer 3）
- 各アイデア・ゾーンに `layer: number` フィールドを保持
- デフォルト: `layer: 1`（Layer 1）

### 4.5 複数選択・一括操作

#### 選択方法

| 操作 | 説明 |
|---|---|
| `Shift+クリック` | アイデアカードをクリックして選択に追加/解除 |
| `Shift+ドラッグ`（空白上） | ラバーバンド選択（矩形内のアイデアを一括選択） |
| `Ctrl+A` | 全アイデアを選択 |
| `f` または `m` キー（moveモード中） | Vimium 風ヒントで追加選択 |

選択中オブジェクトはアウトライン (`move-selected` クラス) でハイライト。

#### 選択中の操作 (`move` モード)

| キー | 動作 |
|---|---|
| `Arrow` | 20px 単位で移動（`Shift+Arrow` で 50px）|
| `g` | 選択中アイデアを重心に集合アニメーション（520ms） |
| `p` | カラーパレットを開き、全選択アイデアに一括色付け |
| `o` | 選択中アイデアの形状を一括トグル（四角 ↔ 丸） |
| `f` / `m` | さらに追加選択 |
| `Enter` | 現在位置で確定（moveモード終了） |
| `Esc` | キャンセル（選択解除） |

#### セレクションツールバー（マウス操作）

複数選択中、画面上部中央にツールバーを表示。マウスだけで以下を実行できる:

| グループ | ボタン | 動作 |
|---|---|---|
| 変形 | `四角` / `丸` | 選択中アイデアの形状を一括設定 |
| コネクタ | `線` / `矢印` / `曲線` / `曲線矢印` | 選択中アイデアにつながっているコネクタ（片端でも可）の種別を一括変更 |

選択が解除されるとツールバーは自動的に非表示。

---

### 4.6 自動レイアウト (Auto Layout)

- `s` キー: 全アイデアに力指向グラフレイアウトを適用
- アルゴリズム: 反発力 (`REPEL = 26000`) + スプリング力 (`SPRING = 0.007`)、180反復
- アニメーション: 700ms、cubic ease-out
- コネクターがある場合: 接続ノードがスプリングで引き合う

---

### 4.7 パン・ズーム

| 操作 | 動作 |
|---|---|
| `Alt+ドラッグ` または 中クリックドラッグ | パン（任意方向） |
| 空白エリアのドラッグ | パン |
| スクロールホイール | ズーム（マウス位置を中心に） |
| `+` / `=` | ズームイン (×1.2) |
| `-` | ズームアウト (÷1.2) |
| `0` | ズームリセット (100%) |
| `Ctrl+Arrow` | パン (80px 単位) |
| `v` → `Enter` | 全オブジェクトが収まるようフィット |
| `v` → `1`〜`9` | テンキー配置の9分割ゾームへ移動 |

ズーム範囲: 25% 〜 250%

---

### 4.8 ミニマップ

- 常時右下に表示（168×112px）
- 全アイデア・ゾーンをスケール描画
- ビューポート枠を青色矩形で表示
- クリックで対応位置へジャンプ
- `Shift+M`: 表示/非表示トグル
- ホバーで不透明度 100%

---

### 4.9 フォーカスナビゲーション

- `Arrow` キー: 現在フォーカスから最も近い方向のオブジェクトへ移動
- `f` キー: Vimium 風ヒント選択（各オブジェクトに A〜Z のラベルが表示）
- `Tab`: 選択モード ↔ 編集モードの切り替え
- `Enter`: 編集モード開始

---

### 4.10 スナップ・アライメント

- グリッドスナップ: 24px 間隔（ドットグリッドに同期）
- `Shift+G`: スナップ ON/OFF 切り替え
- ドラッグ中: 他カードとの左端・右端・中央がオレンジ点線のガイドで表示（6px 以内で検出）
- 重なり解消: 編集確定時にカード間の重なりを自動排除

---

### 4.11 アンドゥ

- `Ctrl+Z` または `u` キー
- 最大 50 ステップ
- 保存対象: `ideas`, `zones`, `connectors`, `nextId`

---

### 4.12 データ入出力

| 操作 | 説明 |
|---|---|
| `JSON` ボタン | 現在の全データを `.json` ファイルとしてダウンロード |
| `読込` ボタン | JSON ファイルをインポートして全体を置換 |
| localStorage 自動保存 | 400ms デバウンスで自動保存（キー: `ideaboard_v4`） |

エクスポート JSON 構造:

```json
{
  "ideas": { ... },
  "connectors": [...],
  "zones": [...],
  "layers": [...],
  "activeLayerId": 1,
  "nextId": 42,
  "meta": { "exported": "ISO8601", "version": 4 }
}
```

### 4.13 コンテキストメニュー（右クリック）

対象を右クリックすると、その場でコンテキストメニューを表示する。

| 右クリック対象 | メニュー項目 |
|---|---|
| アイデア | 編集 / 四角↔丸に変形 / 色を変更 / 複製 / ここから接続 / 削除 |
| ゾーン | ラベル編集 / 色を変更 / リサイズ / 削除 |
| コネクタ（線） | 線 / 矢印 / 曲線 / 曲線矢印 / 削除 |
| 背景 | ここにアイデアを作成 / ゾーンを作成 / 全体にフィット |

- 画面端では自動的に位置を補正して収める
- メニュー外クリック・`Esc`・ホイール操作で閉じる

---

### 4.14 オンボーディング（空ボードガイド）

- アイデアとゾーンが **0 件**のとき、画面中央に主要操作（`i` / `z` / `c` / `o` / 右クリック / `?`）のクイックガイドを表示
- 何か作成されると自動的に消える（`pointer-events: none` で操作の邪魔をしない）
- `?` キーでいつでもヘルプ（ショートカット一覧）の開閉が可能

## 7. アーキテクチャ

### ファイル構成

```
index.html          ← 全コード（CSS + HTML + JavaScript）を1ファイルに内包
SPEC.md             ← 本仕様書
```

### JavaScript 構造

即時実行関数 `App` 内にすべてのロジックを閉じ込め。グローバル汚染なし。

```
App (IIFE)
├── Constants       COLOR_HEX, COLOR_KEYS, GRID_POS
├── State (S)       全ランタイム状態
├── History         pushHistory(), undo()
├── DOM refs        boardWrap, board, zonesLayer, ...
├── Transform       applyXform(), b2s()
├── Minimap         updateMinimap()
├── Mode            setMode()
├── Focus           focus(), clearFocus()
├── Idea            createIdeaAt(), renderIdea(), startEdit()
├── Zone            createZone(), renderZone(), startZoneEdit()
├── Connectors      redrawConnectors(), connectEndpoints()
├── Drag            makeDraggable(), drawGuides()
├── Overlap         resolveOverlaps()
├── Pan/Zoom        wheel, mousedown handlers
├── Keyboard        keydown handler (switch/case)
├── MoveSelection   toggleMoveSelect(), gatherMoveSelection()
├── Viewport        fitAll(), zoomToSection()
├── AutoLayout      autoLayout(), runForceSim()
├── ConnectMode     startConnectMode(), updateConnectLine()
├── HintSelect      openHintSelect(), handleHintKey()
├── Palette         openPalette(), applyColor()
├── LayerSystem     switchLayer(), cycleLayerOpacity(), applyLayerOpacity()
├── Persist         save(), loadSaved()
├── Export/Import   exportJSON(), importJSON()
└── Init            init()
```

### レンダリングモデル

- ボード: `position: absolute` の `#board` 要素（8000×8000px の仮想キャンバス）
- パン・ズーム: CSS `transform: translate() scale()` で全体変換
- コネクター: `position: fixed` の SVG オーバーレイ（スクリーン座標系）
- ゾーン → アイデア の Z 順（ゾーンはアイデアの下）

### ストレージ形式

```
localStorage['ideaboard_v4'] = JSON.stringify({
  ideas, connectors, zones, layers, activeLayerId,
  nextId, pan, zoom, theme
})
```

旧フォーマット (`ideaboard_v3`) からの自動マイグレーション対応。
